## **ECE 3040** - Microelectronic Circuits *(In Progress)*
Semiconductor physics, PN junctions, BJT and MOSFET large/small-signal models, single-stage amplifiers (common-emitter/source), differential amplifiers.

## **ECE 2040** - Circuit Analysis
Node and mesh analysis, superposition, Thevenin and Norton equivalent circuits, operational amplifiers, first- and second-order transient analysis, sinusoidal steady-state analysis using phasors.

## **ECE 2026** - Introduction to Signal Processing
Discrete-time sampling and aliasing, finite and infinite impulse response (FIR/IIR) filters, frequency response, Discrete Fourier Transform (DFT), Z-Transforms.

## **ECE 2031** - Digital Design Laboratory *(In Progress)*
Digital system implementation using FPGAs, Intel Quartus, VHDL logic synthesis, state machine design, timing analysis, simple computer architecture.

## **ECE 2035** - Programming for Hardware/Software Systems
RISC-V assembly, C programming, data structures (linked lists, hash tables), control flow diagrams, memory management (stack/heap).

## **Silicon Jackets** - Analog CMOS IC Design Lectures
Lectures covering analog mixed-signal design fundamentals and MOSFET architecture/physics with hands-on projects including digital inverters, single-stage amplifiers, and ring oscillators in Cadence Virtuoso. Content based on ECE 3400 textbook.
